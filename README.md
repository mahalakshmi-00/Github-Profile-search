# 🔍 GitHub Profile Finder

A simple web app to search GitHub users and display their profiles using the GitHub API.

## ✨ Features

- Search GitHub usernames
- View profile image, name, bio, followers, following, and repos
- Clickable name opens GitHub profile
- Light/Dark mode toggle

## 🚀 Tech Stack

- HTML, CSS, JavaScript
- Axios for API requests
- GitHub REST API
